# Installing Pomodoro

## macOS

### From DMG

1. Open the `.dmg` file
2. Drag **Pomodoro** to your Applications folder
3. **First launch** — macOS will block the app because it's not from the App Store. To open it:
   - **Option A (easiest):** Right-click (or Control-click) the app → click **Open** → click **Open** again in the dialog
   - **Option B (Terminal):** Run `xattr -cr /Applications/Pomodoro.app` then open normally

After the first launch, the app will open normally every time.

### Requirements

- macOS 10.13 or later
- Apple Silicon (M1/M2/M3/M4) — the current build is `aarch64` only
